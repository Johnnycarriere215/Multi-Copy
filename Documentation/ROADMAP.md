# Roadmap

This document outlines the planned development trajectory for Jacque-Copy.

## Vision

To become the best dual clipboard solution on macOS — feeling like a first-party Apple utility while providing a genuinely useful second clipboard that integrates seamlessly into any workflow.

## Release History

- **v1.0.0** — Initial release with dual clipboard, history, search, and full customization ✨ *(Current)*

## Upcoming

### v1.1.0 — Quality of Life
- [ ] Homebrew cask distribution
- [ ] Auto-update improvements with delta updates
- [ ] Localization support (Japanese, Chinese, Korean, Spanish, French, German)
- [ ] Accessibility VoiceOver improvements
- [ ] Keyboard shortcut import/export
- [ ] Clipboard item preview tooltips with rich content

### v1.2.0 — Power Features
- [ ] iCloud sync for settings (opt-in)
- [ ] Clipboard history search via Spotlight
- [ ] Snippet management (predefined clipboard content)
- [ ] Sequential paste mode (paste history items in order)
- [ ] Clipboard filtering by application
- [ ] Custom pasteboard type plugins

### v1.3.0 — Ecosystem
- [ ] Raycast extension for clipboard access
- [ ] Alfred workflow for clipboard access
- [ ] Shortcuts integration (Apple Shortcuts actions)
- [ ] AppleScript/JXA scripting support
- [ ] CLI companion tool (`jc` command)

### v2.0.0 — Platform Evolution
- [ ] iOS companion app with AirDrop clipboard sync
- [ ] iPadOS optimization
- [ ] visionOS clipboard support

## Future Ideas

- Smart clipboard (AI-powered text transformations)
- Clipboard merge/squash operations
- Regex-based clipboard triggers
- Team clipboard sharing (peer-to-peer, encrypted)
- Clipboard versioning/diff

*This roadmap is aspirational and subject to change based on community feedback and maintainer availability.*
